# Kurcoder

A lightweight encoding & decoding tool [BETA v1.0]


### Available Options:
- ROT13 Decoder
- Hexadecimal to ASCII
- ASCII to hexadecimal
- Base64 Encoder
- Base64 Decoder

## Install

__Manual:__
```
$ git clone https://github.com/0xkurome/Kurcoder.git && cd Kurcoder
$ python3 kurcoder.py
```
